Installation
============
There are two ways to use MedSegEvaluator: 1. Install via pip 2. Install from source.

**1. Install from pip (if published):**

.. code-block:: bash

    pip install medsegevaluator


**2. Install from source:**

.. code-block:: bash

    git clone https://github.com/darshandathiya2/MedSegEvaluator.git
    cd MedSegEvaluator
    pip install -e .
